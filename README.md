# clima-promedio-python
Programación Tradicional

En la programación tradicional el programa se organiza mediante funciones y un flujo de ejecución lineal.
Sus características principales son:
El código se centra en las acciones (funciones) que el programa realiza.
La información (temperaturas) se maneja con listas y variables globales o locales.
El programa sigue una estructura secuencial: ingresar datos → procesarlos → mostrar resultados.
Es sencilla, clara y adecuada para programas pequeños o tareas muy específicas.
En esta tarea, este enfoque utilizó funciones separadas para:
Ingresar las temperaturas.
Calcular el promedio.
Ejecutar el flujo principal.


Programación Orientada a Objetos (POO)

La POO organiza el código mediante clases y objetos, lo que permite agrupar datos y comportamientos relacionados.
Sus características principales son:
La información y las acciones se agrupan dentro de una clase.
Facilita conceptos como encapsulamiento, modularidad y posible herencia.
Es ideal para programas más grandes, reutilizables y con crecimiento.
Permite representar entidades del mundo real (en este caso, un objeto "ClimaSemanal").
En esta tarea, la clase incluye:
Un atributo para almacenar las temperaturas.
Métodos para ingresarlas y calcular el promedio.
